TaskTimer Experiment — таймер задач
===================================

Запуск (Linux после .deb): команда timerapp-exp или ярлык в меню.
Python устанавливать не нужно.
Данные (задачи и настройки) сохраняются автоматически в каталоге приложения.

Подробная инструкция: ИНСТРУКЦИЯ.md в репозитории timerapp_exp.


Главный экран
-------------
• Слева — навигация; справа — текущая задача, таймер, Стоп / Завершить.
• Три вида списка: «Сегодня», «В работе», «Все».
• Панель приоритетов 1–4 (фильтр и «Применить» к выделенным).
• Поле даты — задачи, по которым в выбранный день было затрачено время.
• «Сегодня всего» — суммарное время за день.


Работа с задачами
-----------------
• «Новая задача» — название, приоритет, «Добавить» / «Добавить и старт».
• Старт / Стоп, Завершить / Возобновить, История, В план / Из плана.
• Приоритет на день: бейдж 1–4; на «Сегодня» видны задачи с явным приоритетом,
  созданные сегодня или с запущенным таймером.
• Незавершённые из вчерашнего плана переносятся в planned_days, но без приоритета
  на «Сегодня» не показываются — нужен «В план» и выбор приоритета.


Напоминание «продолжать?»
-------------------------
Через заданный интервал (по умолчанию 40 минут) приложение спрашивает, работаете ли вы ещё.
Если не ответить вовремя — таймер остановится. Интервал — в Настройках.


Режим концентрации
------------------
Обратный отсчёт 5–40 минут; рабочая задача ставится на паузу; после окончания можно
продолжить без повторного выбора приоритета.


Трей и плавающий виджет
-----------------------
Окно можно свернуть в трей. Пока идёт задача (или на паузе) — виджет поверх окон.


Битрикс24 (опционально)
-----------------------
Вебхук, импорт с портала, передача времени из истории — см. ИНСТРУКЦИЯ.md.


Технические заметки
-------------------
• Linux: deb amd64, glibc ≥ 2.31. См. docs/system-requirements.md.
• Репозиторий Experiment: github.com/alexandrgert/timerapp_exp
• Версия в ветке: 0.5.51 (timerapp-exp-*.deb).
