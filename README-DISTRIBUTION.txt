TaskTimer — таймер задач для Windows
=====================================

Запуск: дважды щёлкните TaskTimer.exe. Python устанавливать не нужно.
Данные (задачи и настройки) сохраняются автоматически в вашем профиле Windows.

Подробная инструкция (включая Битрикс24): см. ИНСТРУКЦИЯ.md в исходниках или у поставщика.


Главный экран
-------------
• Вкладки «Задачи» и «Фокус». Справа — текущая задача, таймер, кнопки Стоп / Завершить.
• Три вида списка: «План на сегодня», «В работе», «Все задачи».
• Поле даты — задачи, по которым в выбранный день было затрачено время.
• «Сегодня всего» — суммарное время за день.


Работа с задачами
-----------------
• «Новая задача» — создать; «Добавить и старт» — сразу запустить таймер.
• Старт / Стоп, Завершить / Возобновить, История интервалов, В план / Убрать из плана.
• Незавершённые задачи из вчерашнего плана переносятся на сегодня автоматически.


Напоминание «продолжать?»
-------------------------
Через заданный интервал (по умолчанию 40 минут) приложение спрашивает, работаете ли вы ещё.
Если не ответить вовремя — таймер остановится. Интервал — в Настройках (шестерёнка).


Режим «Фокус»
-------------
Обратный отсчёт 5–40 минут, не привязан к задачам.


Трей и плавающий виджет
-----------------------
Окно можно свернуть в трей. Пока идёт задача — маленький виджет поверх окон;
двойной щелчок разворачивает окно.


Битрикс24 (опционально)
-----------------------
1. Входящий вебхук с правами: Задачи, CRM, Пользователи.
2. Настройки → URL вебхука → «Проверить соединение».
3. «Выбрать с портала» — импорт проектов и задач.
4. Завершение связанной задачи синхронизируется с порталом.


Технические заметки
-------------------
• Windows 10/11, 64‑бит (x64). Первый запуск exe может быть чуть медленнее.
• Минимальные требования всех платформ: docs/system-requirements.md в репозитории.
• Перенос на другой ПК: достаточно exe; история на новом компьютере начнётся с нуля
  (или настройте WebDAV для синхронизации).
