## Что нового

### WebDAV — синхронизация между устройствами

- **Отдельные кнопки «Скачать и объединить» / «Загрузить сейчас»** — на десктопе (панель под фильтрами) и в Android (главный экран); вместо одной неоднозначной кнопки облака.
- **Периодическая проверка сервера** — настраиваемый интервал; при изменениях с другого устройства запрос «Скачать и объединить?» / «Позже».
- **«Позже» с таймером** — повтор напоминания через 5 / 10 / 15 / 30 / 60 мин для той же версии на сервере; при новом hash на сервере — сразу новый запрос.
- **Pull-before-push** — перед загрузкой merge с сервером; meta-файл `.sync-meta.json` с SHA-256.
- **Проверка по hash файла** — periodic check скачивает `data.json` и сверяет hash (устаревший meta не скрывает реальные изменения).
- **Merge статуса задачи** — при слиянии учитываются `status` и `completed_at`: завершённая на одном устройстве не теряется; активная сессия → RUNNING.
- **Объединение сессий** — union по `session.id` с сохранением более «богатых» интервалов; корректный разбор Android-меток `+03:00` (время и длительность на десктопе).
- **Android: полноценный WebDAV** — экран настроек, WorkManager в фоне, in-app монитор на переднем плане (без дубля «диалог + уведомление»).
- **Android: минимум 15 мин** для фоновой проверки (ограничение ОС); в открытом приложении — заданный интервал. Подсказка в настройках.

> Пароль WebDAV и вебхук Битрикс24 **не попадают** в облако — только `data.json` с задачами и настройками UI.

### Список задач — десктоп

- **Даты «Создана» / «Завершена»** — read-only в шапке **развёрнутой** карточки (клик по строке); формат `ДД.ММ.ГГГГ ЧЧ:ММ` в **локальном** часовом поясе компьютера.
- **Адаптивная шапка** — при нехватке места мета переносится над счётчиками «сег. · всего».
- **Исправлено наложение UI** — при раскрытой карточке на «Все» и переключении на «Сегодня» / «В работе» строки больше не перекрываются.
- **Стабильная пересборка списка** — без `processEvents()` при смене вкладки/даты.

### Android — задачи

- **Фильтры** — Сегодня / В работе / Все.
- **Вкладка «Все»** — завершённые задачи с зачёркнутым названием.
- **Строки «Создана» / «Завершена»** — в карточке каждой задачи.
- **«Возобновить»** — для завершённых задач (как на десктопе): сброс COMPLETED и старт новой сессии.

> Интеграция с **Битрикс24** на Android пока не реализована (только на десктопе).

### Настройки (десктоп)

- **WebDAV** — расширенная вкладка (интервал, напоминание «Позже», sync on startup/shutdown); увеличена минимальная высота окна настроек.

## Исправления и внутреннее

- **Normalize running** на Android — выбор «победителя» по instant, а не лексикографическому сравнению строк дат.
- **`sync_webdav_now`** — после pull загрузка без повторного download (`push_merged_state`).
- Удалён неиспользуемый API редактирования дат задачи (`parse_task_datetime`, параметры `created_at`/`completed_at` в `update_task`).
- **CI:** проверка совпадения `versionName` Android с `pyproject.toml` перед сборкой APK; GitHub Actions на Node.js 24.
- **`.gitignore`:** keystore, `data.json`, `.gradle-user-home`.
- Расширены тесты: merge status, WebDAV periodic/stale meta, UI карточки, Kotlin DataMerge / TaskRepository.

## Как обновиться

Скачайте нужный файл на [странице релиза v0.5.31](https://github.com/alexandrgert/timer-app/releases/tag/v0.5.31).

### Linux

```bash
wget https://github.com/alexandrgert/timer-app/releases/download/v0.5.31/tasktimer-link-b24-0.5.31-amd64.deb
sudo dpkg -i tasktimer-link-b24-0.5.31-amd64.deb
sudo apt-get install -f
tasktimer-link-b24
```

### Windows

Запустите `tasktimer-link-b24-0.5.31-win64.exe`.

### macOS

Распакуйте `tasktimer-link-b24-0.5.31-macos-arm64.zip` и перетащите `.app` в «Программы».  
*(Сборка CI — Apple Silicon; для Intel Mac нужна отдельная сборка `macos-x86_64`.)*

### Android

```bash
adb install -r tasktimer-link-b24-0.5.31-android.apk
```

Данные (`data.json`) и настройки WebDAV сохраняются при обновлении **поверх** текущей версии с той же подписью APK.  
На каждом устройстве вебхук Битрикс24 настраивается отдельно (не синхронизируется через WebDAV).
