# TaskTimer link B24 — версия 0.3.0

## Что нового

### Синхронизация между компьютерами

Можно хранить базу задач в облаке и работать с одними и теми же задачами на нескольких компьютерах. Подходят сервисы с WebDAV: Nextcloud, Яндекс.Диск (WebDAV), собственный сервер и другие.

В **Настройках → WebDAV** укажите адрес, логин и пароль. Можно включить автоматическую синхронизацию при запуске и при выходе из программы или нажимать «Скачать и объединить» / «Загрузить сейчас» вручную.

Если на двух компьютерах меняли одни и те же данные, программа постарается **объединить** изменения, а не затереть их молча.

### Безопаснее хранятся пароли

Адрес и ключ для Битрикс24 больше **не попадают** в файл задач, который уходит в облако. Пароль от WebDAV тоже хранится отдельно, только на вашем компьютере.

### Удобнее работа из трея

- Плавающий виджет с задачей и временем показывается, пока задача **идёт или на паузе**. После **завершения** задачи виджет **исчезает**.
- При сворачивании окна без активного таймера **не** показываются лишние уведомления.
- Подсказка у иконки в трее показывает названия текущих задач, а не только имя программы.

### Прочие улучшения

- Меню «Выход» и «О программе» (с 0.2.x).
- Более надёжная синхронизация с облаком при нестабильном соединении.
- Исправления при сворачивании окна и завершении задач.

## Кому подойдёт обновление

- Работаете с таймером на **нескольких Linux-компьютерах** и хотите общую базу задач.
- Уже пользуетесь **Битрикс24** — интеграция сохранена, вебхук переносится автоматически при первом запуске новой версии.

---

## Как обновиться

1. Скачайте пакет для Linux (64-bit):

   [tasktimer-link-b24-0.3.0-amd64.deb](https://github.com/alexandrgert/timer-app/releases/download/v0.3.0/tasktimer-link-b24-0.3.0-amd64.deb)

2. Установите:

   ```bash
   sudo dpkg -i tasktimer-link-b24-0.3.0-amd64.deb
   sudo apt-get install -f
   ```

3. Запустите **TaskTimer link B24** из меню или командой `tasktimer-link-b24`.

**Ваши задачи и история времени сохранятся.** Настройки Битрикс24 подхватятся с первого запуска.

Чтобы включить облако: **шестерёнка → WebDAV** — заполните поля и при необходимости нажмите «Проверить соединение».

---

## На что обратить внимание

- Не запускайте **одну и ту же задачу одновременно** на двух компьютерах — при слиянии данных может остаться только одна копия интервалов времени.
- Облако хранит **задачи и настройки интерфейса**, но **не** пароли — их нужно настроить на каждом компьютере (или один раз перенести файл настроек вручную).
