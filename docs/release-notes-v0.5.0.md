# TaskTimer link B24 — версия 0.5.0

## Что нового

### Задачи и сессии

- **Комментарий к сессии** — при добавлении или правке интервала можно указать текст; он сохраняется в `data.json` и подставляется при передаче времени в Битрикс24.
- **Редактирование задачи** — кнопка «Изменить» в строке задачи: можно поменять название и описание без пересоздания.

### Панель таймера

- **Светлая тема** — панель таймера оформлена в светлых тонах, как основное окно.
- **Описание при работе** — пока таймер идёт, на панели видно описание активной задачи.
- Элементы выровнены **по верхнему краю** — удобнее на небольших экранах.

### Трей (из 0.4.5)

- При сворачивании окна в подсказке иконки снова видны **названия текущих задач**, а не только имя приложения.

## Как обновиться

### Linux

```bash
wget https://github.com/alexandrgert/timer-app/releases/download/v0.5.0/tasktimer-link-b24-0.5.0-amd64.deb
sudo dpkg -i tasktimer-link-b24-0.5.0-amd64.deb
sudo apt-get install -f
```

### Windows / macOS / Android

Скачайте нужный файл на [странице релиза v0.5.0](https://github.com/alexandrgert/timer-app/releases/tag/v0.5.0).

Данные (`data.json`) и настройки в `~/.config/tasktimer/` сохраняются при обновлении.
