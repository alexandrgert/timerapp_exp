Исправления стабильности и Битрикс24. Рекомендуется обновление с **0.4.3** и ниже.

**Битрикс24**
- Передача времени в проект работает на разных порталах (поля журнала определяются автоматически).
- Короткие интервалы не обнуляются при отправке в проект.
- Повторная передача после сбоя не создаёт дубль на портале.

**Интерфейс**
- Плавающий виджет: кнопка **✕** и пункт трея **«Показать виджет»**.
- Панель таймера показывает последнюю приостановленную задачу (**«Продолжить»**).

**Старт**
- Синхронизация WebDAV не блокирует появление окна.
