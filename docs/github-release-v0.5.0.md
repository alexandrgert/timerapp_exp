Новые возможности для учёта времени и удобнее панель таймера. Рекомендуется обновление с **0.4.5** и ниже.

**Задачи и сессии**
- К каждой сессии можно добавить **комментарий** — он сохраняется и передаётся в Битрикс24 при выгрузке времени.
- Задачу можно **отредактировать** (название и описание) кнопкой «Изменить» в списке.

**Панель таймера**
- Светлая тема, описание задачи видно **пока таймер идёт**.
- Элементы выровнены по верхнему краю.

**Трей**
- При сворачивании окна в подсказке снова видны названия активных задач.
